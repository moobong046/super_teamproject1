# 🎬 IMDB Top 1000 영화 평점 데이터 분석 (Numpy)

## 프로젝트 개요
IMDB의 Top 1000 영화 데이터셋(`IMDB top 1000.csv`)을 활용하여 영화들의 평점 분포, 장르별 특성, 그리고 연도별 평점 트렌드를 분석한 프로젝트입니다. Pandas 라이브러리 없이 순수 **Numpy**를 활용하여 데이터 전처리 및 통계 분석을 수행하며 데이터 핸들링의 기초 역량을 다졌습니다.

## 기술 스택 (Tech Stack)
* **Language:** Python
* **Data Manipulation:** Numpy, CSV
* **Data Visualization:** Matplotlib

## 주요 분석 로직
1. **데이터 전처리:** `csv` 모듈로 데이터를 로드하고 Numpy 배열로 변환. 조건부 마스킹(`~np.any`)을 통해 빈 값이 있는 결측치 데이터 필터링.
2. **문자열 파싱:** 영화 제목(`Title`) 컬럼에서 문자열 슬라이싱(`[-5:-1]`)을 통해 개봉 연도(Year) 추출.
3. **통계 연산:** Numpy의 `mean()`, `max()`, `min()` 함수를 활용하여 전체 및 조건별(장르, 연도) 통계치 계산.

## 📊 주요 분석 결과

### 1. 전체 데이터 요약
* **분석 대상 영화 수:** 1000편
* **전체 평균 평점:** 8.1점
* **최고 평점 / 최저 평점:** 9.3점 / 8.0점
* 🏆 **최고 평점 영화:** *The Shawshank Redemption (1994) - 9.3점*

### 2. 장르별 평균 평점 (Top Insights)
총 21개의 장르 조합을 분석한 결과, 전반적으로 8.1~8.2점의 고른 분포를 보였습니다.
* **최고 평점 장르:** Western (8.4점)
* **주요 장르 평점:** Drama (8.1), Action (8.1), Horror (8.2), Sci-Fi (8.1)

### 3. 연도별 평점 트렌드
1920년대부터 2020년까지의 개봉 연도별 평균 평점을 분석하고 선 그래프(Line Chart)로 시각화했습니다. 
* 특정 연도(예: 1946년, 1974년 8.6점 / 1990년, 2020년 8.7점)에 명작들이 집중되어 평점이 높게 튀는 경향성을 확인했습니다.
